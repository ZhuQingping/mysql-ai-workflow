# /workflow-review

Review a design, patch, task report, or generated workflow artifact.

Read:

- `<workflow-repo>/common/review-gate.md`
- `common/verification-matrix.md`
- selected scenario workflow
- selected domain profile review gate, when relevant

Output findings first:

- Critical
- Important
- Minor

Then include:

- open questions
- verification evidence
- recommendation: ACCEPT, ACCEPT WITH RISKS, REVISE, or REJECT
