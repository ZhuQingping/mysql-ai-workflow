# AI Workflow Repository Context

This repository contains reusable, agent-neutral workflow documents.

Start with:

- `README.md`
- `design.md`
- the relevant file under `scenarios/`
- the relevant file under `common/`
- the relevant tool adapter under `adapters/`
- the relevant domain profile under `profiles/`, if any

Do not treat a tool adapter as the workflow source of truth. Adapters explain
how a tool applies the common workflow.
